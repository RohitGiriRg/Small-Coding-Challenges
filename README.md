# 🚀 Small Coding Challenges

A collection of small, fun, and beginner-friendly coding challenges built using **JavaScript** and **TypeScript**.

## 🛠️ Requirements

Make sure you have the following installed:

1. **Node.js & npm**
   👉 Download from the official website: https://nodejs.org/

2. **TypeScript**
   Install globally:

   ```bash
   npm install -g typescript
   ```

3. **Java (JDK)** *(optional for some problems)*
   👉 Download from the official website: https://www.oracle.com/java/

---

## ▶️ How to Run

### Run JavaScript files

```bash
node filename.js
```

### Run TypeScript files

```bash
tsc filename.ts
node filename.js
```
